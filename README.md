# 🔐 Random Password Generator

**Автор:** Магомедова Хамис Исаевна

---

## 📌 О проекте

Консольное и GUI-приложение для генерации надёжных паролей.  
Позволяет гибко настраивать состав пароля, сохранять историю и управлять ею.

---

## 🚀 Возможности

- выбор длины пароля (от 4 до 32 символов);
- включение/отключение цифр, букв и спецсимволов;
- генерация случайного пароля одной кнопкой;
- копирование пароля в буфер обмена;
- автоматическое сохранение истории в JSON;
- просмотр истории в таблице;
- очистка всей истории.

---

## 🖥️ Внешний вид

Приложение состоит из трёх блоков:

1. **Панель настроек** — ползунок длины и чекбоксы.
2. **Поле результата** — сгенерированный пароль и кнопка копирования.
3. **Таблица истории** — список всех созданных паролей с датой и параметрами.

Внизу таблицы расположена кнопка очистки истории.
